# 🎓 Smart Campus Helper

A modern, responsive React web app that helps university students navigate campus life — find study spots, explore campus services, discover events, and access important contacts.

---

## ✨ Features

- 🔍 Search and filter campus services by category
- 📍 Browse and filter study spots by type and availability
- 📅 View upcoming campus events
- ❓ FAQ accordion with instant answers
- 📬 Contact form with validation
- 📱 Fully responsive — works on mobile, tablet, and desktop
- ⚡ Fast and lightweight — no backend required

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React 19 | Frontend framework |
| React Router DOM | Client-side routing |
| Tailwind CSS | Utility-first styling |
| Vite | Build tool and dev server |
| Google Fonts | Typography (Sora + DM Sans) |

## 🚀 Getting Started

### Prerequisites
- Node.js v18+
- npm or yarn

### Installation

```bash
git clone https://github.com/khodoribrahim313/smart-campus.git
cd smart-campus-helper
npm install
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
npm run preview
```
